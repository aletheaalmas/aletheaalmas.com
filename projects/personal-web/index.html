My First Personal Website

About This Website & My Learning Journey

This is my first personal website, built from scratch using HTML and CSS. I created it to showcase my projects and share my journey in learning web development.


After being laid off in April 2025, I started looking for new job opportunities and applied to many companies, but things didn’t work out. Then i decide to learn a new skill "web development" while also looking for new job. I began by teaching myself HTML through FreeCodeCamp, but soon realized that I needed proper guidance. That’s when I joined a web development mentoring program at Bearmentor. My mentor guided me step by step, since I had no prior background in this field. I learned how to use Git, the terminal, code editors, Figma, and much more.


I started designing the website layout using Figma, looking for inspiration online, creating wireframes, components, and designing each page carefully.

Once the design was ready, I began coding it with HTML and CSS. It took me some time to make the layout match the design, but in the process I learned a lot about CSS properties, flexbox, grid, and responsive design.


After 1 month of learning, designing, and coding, I was finally able to complete this website. Although it’s still far from perfect and there’s so much more to learn, I plan to keep improving both the features and the design over time.
